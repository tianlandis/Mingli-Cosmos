## 3.1 六冲（6 Pairs of Clash · Chong）

```
子⇔午  丑⇔未  寅⇔申  卯⇔酉  辰⇔戌  巳⇔亥
```

> 冲 = 相克 + 方向相对，力量最强。

|:---:|:---:|

| 寅亥 | 木 |
| 卯戌 | 火 |
| 辰酉 | 金 |
| 巳申 | 水 |
| 午未 | 土 |

---

## 3.3 三合局（4 Three-Unions · San He）

| 合局 | 化出 | 半合（任意两字） |
|:---:|:---:|:---:|
| 申子辰 | 水 | 申子 / 子辰 / 申辰 |
| 寅午戌 | 火 | 寅午 / 午戌 / 寅戌 |
| 亥卯未 | 木 | 亥卯 / 卯未 / 亥未 |
| 巳酉丑 | 金 | 巳酉 / 酉丑 / 巳丑 |

> 半合力量弱于全合，但仍有影响。

---

## 3.4 三会局（4 Three-Meetings · San Hui）

```
亥子丑 → 水  (北方)
寅卯辰 → 木  (东方)
巳午未 → 火  (南方)
申酉戌 → 金  (西方)
```

> 三会局力量 > 三合局，是地支最强合局。

---

## 3.5 相刑（Punishment · Xing，含自刑）

| 类型 | 组合 | 名称 |
|------|------|------|
| 子⇔卯 | 互刑 | 无礼之刑 |
| 寅⇔巳⇔申 | 三刑 | 无恩之刑 |
| 丑⇔戌⇔未 | 三刑 | 持势之刑 |
| 辰/午/酉/亥 | 自刑 | 自刑 |

> 自刑：地支与自己相同即构成（辰辰、午午、酉酉、亥亥）。

## 3.8 空亡（Void · Kong Wang，六甲旬空亡）

根据**日柱干支**查所属六甲旬：

| 六甲旬 | 空亡地支 |
|--------|:------:|
| 甲子旬 (甲子→癸酉) | 戌、亥 |
| 甲戌旬 (甲戌→癸未) | 申、酉 |
| 甲申旬 (甲申→癸巳) | 午、未 |
| 甲午旬 (甲午→癸卯) | 辰、巳 |
| 甲辰旬 (甲辰→癸丑) | 寅、卯 |
| 甲寅旬 (甲寅→癸亥) | 子、丑 |

**规则**: 四柱中任一地支落入空亡，该柱力量减弱。日支落空亡则配偶宫虚化。
