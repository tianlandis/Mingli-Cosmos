## 2.1 相生链（Generating Cycle）

```
金 → 水 → 木 → 火 → 土 → 金
```

口诀：金生水，水生木，木生火，火生土，土生金。

---

## 2.2 相克链（Overcoming Cycle）

```
金 → 木 → 土 → 水 → 火 → 金
```

口诀：金克木，木克土，土克水，水克火，火克金。

---

## 2.3 十神判断规则（Ten Gods Judgment）

以 **日主天干** 为参照，对比目标天干：

```
五行关系 + 阴阳异同 → 十神
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
同五行 + 同阴阳  → 比肩
同五行 + 异阴阳  → 劫财
我生   + 同阴阳  → 食神
我生   + 异阴阳  → 伤官
生我   + 同阴阳  → 偏印
生我   + 异阴阳  → 正印
我克   + 同阴阳  → 偏财
我克   + 异阴阳  → 正财
克我   + 同阴阳  → 偏官（七杀）
克我   + 异阴阳  → 正官
```

> 记忆法：先看五行关系（同/生/被生/克/被克），再看阴阳异同（同→偏/比/食，异→正/劫/伤）。
