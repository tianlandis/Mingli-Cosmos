## 一、V2.0 取格决策树

```
月令地支类型?
├── 四正月（子午卯酉）──→ 直接取本气十神 → 定格
│   └── 藏干唯一，无需争议
│
├── 杂气月 / 四长生月
│   ├── 月令藏干透出天干? (优先级: 月干 > 年干 > 时干)
│   │   ├── 是 → 取透出藏干十神 → 定格（传统子平·透干取用）
│   │   │      └── 比劫透干? → 直接立建禄格/羊刃格（不做退位）
│   │   └── 否 → 启动月令分金算法 ↓
│   │
│   └── 【月令分金 (FenJin)】
│       ├── 阳男阴女(大运顺排) → 分金逆数 → effectiveDays = 30 - qiYunDays
│       ├── 阴男阳女(大运逆排) → 分金顺数 → effectiveDays = qiYunDays
│       ├── 在藏干天数刻度尺上定位 → 找到对应藏干
│       └── 结果十神?
│           ├── 非比劫 → 直接定格局 ✓
│           └── 比劫 → 比劫不成格处理
│               ├── 禄格破例? (四长生月 + 日干得禄) → 建禄格
│               ├── 羊刃格破例? (四正位 + 日干羊刃) → 羊刃格
│               └── 通用规则 → 进退位取邻干，跳过比劫继续移
│                   └── 所有位皆比劫? → 退取月令本气
```

## 二、起运天数计算（QiYunDays）

> 实现位置: `calculator.ts` → `computeQiYunDays()`

| 条件 | 大运方向 | 节气计算 |
|------|:---:|------|
| 阳年男 / 阴年女 | 顺排 | 出生日到**下一个**节气的天数 |
| 阴年男 / 阳年女 | 逆排 | 出生日到**上一个**节气的天数 |

使用 `lunar.getNextJie()` / `lunar.getPrevJie()` 精确计算儒略日差。

换算规则：**3天 = 1岁起运**，1天 = 4个月，1时辰 ≈ 10天。

## 四、格局组合判定（7种）

在确定主导格局后，检查天干中是否存在关键十神组合：

| 组合名 | 条件 | 含义 |
|------|------|------|
| **官印相生** | 印格 + 正官透出 | 印得官生，贵气注入 |
| **煞印相生** | 印格 + 七杀透出 | 杀印相生，化杀为权 |
| **财官相生** | 财格 + 正官透出 | 财生官，富贵之道 |
| **食神生财** | 食神格 + 财星透出 | 食神生财，技艺致富 |
| **伤官生财** | 伤官格 + 财星透出 | 伤官生财，才华变现 |
| **伤官佩印** | 伤官格 + 印星透出 | 印制伤官，正反回调 |
| **纯格** | 以上条件均不满足 | 无关键组合，为纯格 |

## 六、破格风险检测（5类）

| 风险类型 | 检测条件 | 严重度 | MBTI补益 |
|------|------|:---:|------|
| **伤官见官** | 伤官+正官同时透出 | 高 | 强化 Ti（伤官格）或 Ni（官格） |
| **财星破印** | 财星+印星并存且格局为印格 | 中 | 向 ESTJ(Te+Si) 靠拢 |
| **官杀混杂** | 正官+七杀各≥1 | 中 | 明确 Te 或 Ti 一方为主导 |
| **印格无官杀** | 纯印格，无官杀配合 | 中 | 向 ENTJ(Te+Ni) 靠拢 |
| **财格无食伤** | 纯财格，无食伤配合 | 低 | 向 ESFP(Se+Fi) 靠拢 |

## 七、吉凶格分类与基本喜忌

| 格局类型 | 吉凶属性 | 基本喜忌 |
|---------|---------|---------|
| 正官、正印、偏印、食神、正财、偏财 | **吉格（喜生扶）** | 喜大运、流年生扶该十神，最忌被克制、混杂、合坏 |
| 七杀、伤官、羊刃 | **凶格（喜制化）** | 七杀/伤官：喜见印（化）或食神/正印（制）。羊刃：喜见七杀（驾），或食伤（泄），或印（化） |
| 禄格 | **中性（偏吉）** | 喜顺局（原局或大运有财官印食引导），**最忌冲禄** |

> **格局无吉凶之分，只有成败之别**：成格 = 能量纯粹、流通顺畅、有护卫；败格 = 能量混杂（官杀混杂、食伤混杂）、被冲破、压制无制。
