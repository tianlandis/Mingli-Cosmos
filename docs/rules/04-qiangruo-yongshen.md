## 一、日主强弱评分（Day Master Strength）

### 4.1 评分模型（总分 0-100）

```
总得分 = 月令分 + min(地支根气分, 40) + min(比劫分 + 偏印分×0.6, 25)
       ± 全局平衡调整
       + 强根加分 (最多+10)
最终 clamp 到 [0, 100]
```

### 4.2 各维度细则

| 维度 | 得分规则 |
|------|---------|
| **月令** | 旺=40, 相=25, 余气=10, 休=0, 囚=0 |
| **地支根气** | 本气=15, 同五行=5, 藏干每个=5；×权重(日支1.2 > 月支1 > 时支0.8 > 年支0.5) → 累加后 cap 40 |
| **天干比劫** | 年/月/时干每有一个比劫 → +10 |
| **天干生扶** | 比劫+10 + 印星+8 |
| **天干克泄耗** | 官杀+8 + 食伤+7 + 财+6 |
| **全局调整** | 克泄耗 > 生扶×2 → -10；生扶 > 克泄耗×2 → +10 |
| **强根加分** | 日支本气根 +5；月令本气且当令 +5 |

### 4.3 等级映射

| 分数 | 等级 | 用神策略 |
|:---:|------|------|
| ≥75 | 极强 | 取克泄耗 |
| ≥60 | 强 | 取克泄耗 |
| ≥48 | 中和偏强 | 取克泄耗 |
| ≥38 | 中和 | 均衡+调候 |
| ≥28 | 中和偏弱 | 取生扶 |
| ≥15 | 弱 | 取生扶 |
| <15 | 极弱 | 取生扶（或从格） |

## 二、用神忌神（Useful & Unfavorable Gods）

### 5.1 取用规则

| 日主状态 | 用神（喜） | 忌神 | 逻辑 |
|---------|-----------|------|------|
| 身强 | 克泄耗（官杀+食伤+财） | 生扶（印+比劫） | 制强求平衡 |
| 身弱 | 生扶（印+比劫） | 克泄耗（官杀+食伤+财） | 扶弱求平衡 |
| 中和 | 日主 + 印星 | 无明确忌 | 以调候为先 |

### 5.2 调候规则（Climate Adjustment）

| 月令 | 调候五行 | 说明 |
|------|:---:|------|
| 巳/午/未 (夏) | 水 | 火炎土燥，喜水润局 |
| 亥/子/丑 (冬) | 火 | 水冷金寒，喜火暖局 |
| 申/酉/戌 (秋) + 日主金 | 火 | 金旺喜火炼 |

> ⚠️ **注意**: 调候五行自动追加到用神列表中，不替代原有用神。

### 5.3 全局偏向调整

- 统计全命局（四柱干支+藏干）各五行出现次数
- 判断全局主导五行和最弱五行
- 用于辅助修正用神忌神选择
