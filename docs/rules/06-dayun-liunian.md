## 一、大运分析（DaYun Analysis）

### 7.1 大运好坏判断

```
天干五行 = 用神 → +1.5 分
天干五行 = 忌神 → -1 分
地支五行 = 用神 → +1 分
地支五行 = 忌神 → -1 分

总分 ≥ 2  → 佳
总分 ≤ -2 → 不佳
其他      → 平
```

### 7.2 大运判断三要素

1. **看大运地支与日干的十二长生关系** — 反映日主在该阶段的基本生命力状态
2. **看大运地支与格局之间的生克关系** — 判断吉凶核心（生扶格局=好运，克制格局=坏运）
3. **看大运地支与日干本身的生克关系** — 辅助参考

> **大运重地支，轻天干**。天干主要论"合"（甲己合、乙庚合等）。

## 二、流年太岁分析（Annual · TaiSui，6种）

| 关系 | 条件 | 提示 |
|------|------|------|
| **值太岁** | 流年支 = 年支 | 本命年，宜低调行事 |
| **冲太岁** | 流年支 冲 年支 | 变动较大，注意人际关系 |
| **合太岁** | 流年支 合 年支 | 人缘佳，适合合作 |
| **刑太岁** | 流年支 刑 年支 | 防口舌是非和法律问题 |
| **破太岁** | 流年支 破 年支 | 防小人暗中破坏 |
| **害太岁** | 流年支 害 年支 | 防背后中伤和慢性病 |

> **流年重天干，轻地支**。流年天干主要看与格局、大运的合化关系及"伤官见官"等关键组合；地支主要论神煞和刑冲合害。

## 四、六大专题批注（Special Topics）

### 8.1 性格

- 日主五行特性（基础）
- 十神主导性格（按数量取前 3 个十神）
- 日支五行特性

### 8.2 事业

- 十神统计判断（官杀≥3 → 体制内；印≥3 → 学术；食伤≥3 → 创意；财≥3 → 经商）
- 特殊组合：官印相生、食伤生财
- 日主五行 → 行业方向

### 8.3 财运

- 正财/偏财数量 → 收入模式
- 食伤生财 → 技术致富
- 比劫多 + 财星 → 破财风险

### 8.4 婚姻

| 性别 | 判断依据 |
|:---:|------|
| 男 | 财星（正财+偏财）数量，财多→异性缘佳，财无→晚婚 |
| 女 | 官杀（正官+偏官）数量，官杀混杂→选择多，官无→晚婚 |

**配偶宫（日支）分析（6种）**:

| 关系 | 条件 | 含义 |
|------|------|------|
| 逢冲 | 月支或时支 冲 日支 | 聚少离多，需用心经营 |
| 逢合 | 月支或时支 合 日支 | 伴侣人缘佳，需沟通维系 |
| 逢刑 | 月支或时支 刑 日支 | 多摩擦，需互相包容 |
| 被害 | 月支或时支 害 日支 | 易受外界干扰 |
| 被破 | 月支或时支 破 日支 | 感情基础不稳固 |
| 空亡 | 日支落入空亡 | 缘分较淡，宜晚婚 |

### 8.5 健康

- 五行缺失 → 对应器官需保养（木=肝胆, 火=心血管, 土=脾胃, 金=肺, 水=肾）
- 五行过旺（≥5个） → 对应器官易有负担
- 日主偏弱 → 需增强体质

### 8.6 子女

| 性别 | 判断依据 |
|:---:|------|
| 女 | 食伤（食神+伤官）数量 |
| 男 | 官杀（正官+偏官）数量 |

> 时柱为子女宫，附五行分析。
