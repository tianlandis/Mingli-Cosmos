## Phase 1：结构化批注 + AI 增强 🔥 当前重点

### 已完成 ✅

- [x] 规则引擎批注：日主强弱、用神忌神
- [x] **V2.0 格局判断**（2026-06-16 完成）
  - 四正月直接取格
  - 杂气月透干取用（月干 > 年干 > 时干）
  - 月令分金算法（藏干天数刻度尺 + 起运天数 + 分金方向）
  - 比劫不成格处理（禄格/羊刃格破例 + 进退位）
  - 格局组合判定（7 种组合 + 纯格检测）
  - 破格风险检测（4 类风险 + MBTI 补益建议）
  - MBTI 人格映射（十神→认知功能 + 画像 + 行业适配 + 能量调整）
- [x] 神煞规则（天乙贵人/文昌/桃花/驿马/华盖/金舆/禄/羊刃 等 8 种）
- [x] 专题批注：婚姻、事业、健康、财运、性格、人际
- [x] 地支关系全面分析
- [x] `engine/pattern/` 格局引擎子目录
- [x] `engine/rules/` 神煞规则模块

### 待完成 ⬜

- [ ] 大运流年深度解读（当前为基础版）
- [ ] 10+ 历史名人八字验证测试

## Phase 2：AI 深度集成 ⬜ 待启动

- [ ] **AI 智能助手**（核心交付）
  - 多轮对话（"今年财运如何？""适合什么职业？"）
  - 规则 JSON 作为事实护栏，禁止 AI 编造
- [ ] **提示词模板库** (`aiPrompts.ts`)
- [ ] **AI 对话组件** (`AIChat.tsx` + `useChat.ts`)
- [ ] **报告生成** (`ReportGenerator.tsx` — PDF + 长图)
- [ ] 模拟预测（"如果换城市发展会怎样？"）
- [ ] 合婚、择吉等扩展功能

> ⚠️ Phase 2 所有模块仅通过 `contracts/engine-api.md` 消费核心引擎，不修改 `engine/` 内部代码。
