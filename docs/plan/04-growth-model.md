# 04-增长与变现模型（Growth & Monetization Model）

> **来源**: `ProjectPlan.md` §八 | **用途**: 运营/推广/商业决策参考
> **受众**: 运营 / 市场 / 产品经理

---

## 增长渠道

| 渠道 | 内容策略 | 优先级 |
|------|----------|:---:|
| **小红书** | 真实案例 + 2026 运势解读 | 高 |
| **抖音 / B 站** | 排盘过程短视频 + 趣味命理 | 中 |
| **SEO** | 针对"AI 八字排盘""精准批命"等关键词 | 中 |
| **病毒传播** | 一键生成精美分享图 | 高 |

---

## 变现模式（Freemium）

| 等级 | 内容 | 定价 |
|------|------|------|
| **免费** | 基础排盘 + 简单批注 + 有限 AI 对话 | ¥0 |
| **付费订阅** | 无限 AI 对话 + 深度报告 + 历史记录 | ¥9.9-29/月 |
| **单次购买** | 单次深度报告 | ¥3.9-9.9 |
| **企业合作** | 心理咨询合作 / API 输出 | 协商 |

---

## 营销亮点

| 亮点 | 说明 |
|------|------|
| **差异化定位** | "传统规则 + AI 智慧"，非泛泛 AI 算命 |
| **隐私保护** | 本地计算优先，不强制上传敏感数据 |
| **文化尊重** | 规范引用经典原文，拒绝过度玄幻 |
