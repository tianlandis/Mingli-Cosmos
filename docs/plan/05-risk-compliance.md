# 05-风险与合规（Risk & Compliance）

> **来源**: `ProjectPlan.md` §九 | **用途**: 合规审查、法务参考
> **受众**: 全员 / 法务

---

## 风险矩阵

| 风险类别 | 具体风险 | 应对策略 | 等级 |
|----------|----------|----------|:---:|
| **伦理** | 用户过度依赖命理决策 | 每份报告添加醒目免责："非科学预测，仅供娱乐参考" | 🔴 高 |
| **准确性** | 规则引擎输出错误 | 规则为主，AI 为辅；持续用历史案例校验 | 🟡 中 |
| **法律合规** | 涉及医疗/法律/金融建议 | 避免医疗/法律/金融等领域绝对化建议 | 🔴 高 |
| **LLM 成本** | API 调用费用过高 | 缓存常见问题，使用高性价比模型 | 🟡 中 |
| **时辰精度** | 时辰不准确导致排盘错误 | 默认使用北京时间，后期支持真太阳时 | 🟢 低 |
| **数据隐私** | 用户生辰数据泄露 | 本地计算优先，不强制存储敏感信息 | 🔴 高 |

---
