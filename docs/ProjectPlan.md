# 八字排盘系统 — 项目长期计划书（2026 AI增强版）


## 一、项目概述

| 项目 | 内容 |
|------|------|
| **项目名称** | 八字排盘 & AI智能命理助手（Bazi AI / 八字智解） |
| **产品定位** | 精准、专业、好看的新中式AI八字命理工具 |
| **核心理念** | 传统子平法为核心，现代AI技术为辅助 — **计算准确 + 规则透明 + 文化尊重** |
| **目标用户** | 25-45 岁都市年轻群体（事业/婚姻/人生焦虑者）、传统文化爱好者 |

### 产品形态演进

```
阶段 1：Web 网页  ──→  阶段 2：PWA + 微信小程序  ──→  阶段 3：独立 App + 生态社区

## 三、技术方案

### 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | React 19 + TypeScript 6 + Vite 8 + Tailwind CSS 4 |
| **核心计算** | `lunar-typescript`（6tail） |
| **规则引擎** | 纯 TypeScript 函数，零外部依赖 |
| **AI 集成** | Vercel AI SDK / LangChain + DeepSeek / Claude / Grok |
| **其他** | localStorage 缓存、PWA 支持、Playwright 测试 |

```

│                  UI 层 (React)                    │
│  BaziChart │ AnnotationPanel │ AIChat │ Report    │

- **优先级：Phase 2.1 → 2.4 顺序实现**


## 五、AI 增强方向

### AI 核心机制

```
用户问题 + 用户背景
        ↓
规则引擎输出（结构化 JSON）──→ AI Prompt（护栏）
        ↓
  LLM 生成回复（自然语言）
```

- **输入**：规则引擎 JSON + 用户背景 + 具体问题
- **Prompt**：专业提示词模板（CoT、Few-shot、经典引用约束）
- **输出控制**：规则 JSON 作为事实护栏，禁止 AI 编造未验证内容

### AI 具体功能

| 功能 | 说明 |
|------|------|
| **智能对话助手** | 多轮对话，记忆上下文 |
| **个性化报告生成** | 一键生成，支持导出 PDF / 长图 |
| **场景模拟** | 决策辅助（"换城市发展""创业时机"等） |
| **RAG 知识库** | 后期接入《滴天髓》等经典文献 |
| **多模态可视化** | AI 生成五行能量图、运势路径图 |

### AI 集成技术

- **前端**：Vercel AI SDK 或 OpenAI-compatible API
- **后端可选**：Node.js / Next.js API Routes
- **成本控制**：缓存常见问题 + 使用性价比高的模型

| 等级 | 内容 | 定价 |

### 营销亮点

- **差异化定位**："传统规则 + AI 智慧"，非泛泛 AI 算命
- **隐私保护**：本地计算优先，不强制上传敏感数据
- **文化尊重**：规范引用经典原文，拒绝过度玄幻


## 九、风险与注意事项

| 风险类别 | 应对策略 |
|----------|----------|
| **伦理声明** | 每份报告添加醒目免责："非科学预测，仅供娱乐参考" |
| **准确性** | 规则为主，AI 为辅；持续用历史案例校验 |
| **法律合规** | 避免医疗/法律/金融等领域绝对化建议 |
| **LLM 成本** | 缓存常见问题，使用高性价比模型 |
| **时辰精确度** | 默认使用北京时间，后期支持真太阳时 |
| **数据隐私** | 本地计算优先，不强制存储敏感信息 |
