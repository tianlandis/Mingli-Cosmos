# 🎨 UI/UX 设计规范（Design Specification）

> **来源**: `ProjectPlan.md` §六 | **用途**: UI 组件开发、视觉改版时的设计依据
> **受众**: 前端开发 / UI 设计师

---

## 设计风格

**新中式 + 现代简约**

---

## 配色系统（Color Palette）

| 色值 | 名称 | 用途 |
|------|------|------|
| `#1a1410` | 深棕黑底 | 主背景色 |
| `#c41e3a` | 朱红 | 强调色 / 主色调 |
| `#d4a853` | 鎏金 | 装饰 / 高亮 |
| `#e7d7c0` | 暖米色 | 正文 / 卡片背景 |

> 在 Tailwind CSS v4 中通过 `@theme` 指令定义 CSS 变量。

---

## 字体（Typography）

| 优先级 | 字体 | 用途 |
|:---:|------|------|
| 1 | Microsoft YaHei | Windows 默认 |
| 2 | PingFang SC | macOS / iOS |
| 3 | Noto Sans SC | Linux / 备选 |

字号层级: 标题 (text-xl) → 正文 (text-base) → 批注 (text-sm) → 辅助 (text-xs)

---

## 布局（Layout）

**单页应用，上下滚动**，无多页面路由。

```
┌──────────────────────────────┐
│         Header 头部           │
├──────────────────────────────┤
│      BirthForm 输入区         │
├──────────────────────────────┤
│      BaziChart 排盘结果       │
│   (四柱 + 藏干 + 纳音)        │
├──────────────────────────────┤
│   FiveElements 五行可视化     │
│   TenGods 十神分析            │
├──────────────────────────────┤
│   AnnotationPanel 批注面板    │
│   (可折叠专题 + MBTI + 风险)  │
├──────────────────────────────┤
│   LuckCycle 大运流年时间线    │
├──────────────────────────────┤
│   AI Chat 浮窗 (Phase 2)     │
└──────────────────────────────┘
```

---

## 核心页面流（User Flow）

```
输入区 → 排盘结果 → 批注面板（可折叠专题 + MBTI画像 + 风险提示） → AI 聊天浮窗
```

## 可视化元素

| 元素 | 说明 | 状态 |
|------|------|:---:|
| 五行进度条 / 环形图 | 五行分布可视化 | ✅ |
| 日主强弱评分 | 得分 + 等级展示 | ✅ |
| 大运时间线 | 十年一运横向滚动 | ✅ |
| 分享卡片 / 海报 | 一键生成精美分享图 | ⬜ Phase 3 |

---

## 组件规范

- 每个组件文件 ≤ 300 行，超出则拆分子组件
- 组件 Props 必须定义 TypeScript 接口
- 样式使用 Tailwind CSS 原子类，避免内联 style
- 所有颜色通过 Tailwind theme 引用，不硬编码色值
