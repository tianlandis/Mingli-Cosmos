```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
同五行 + 同阴阳  → 比肩
同五行 + 异阴阳  → 劫财
我生   + 同阴阳  → 食神
我生   + 异阴阳  → 伤官
生我   + 同阴阳  → 偏印
生我   + 异阴阳  → 正印
我克   + 同阴阳  → 偏财
我克   + 异阴阳  → 正财
克我   + 同阴阳  → 偏官（七杀）
克我   + 异阴阳  → 正官
```

## 4. 日主强弱评分

> 实现位置: `annotation/dayMasterStrength.ts`

### 4.1 评分模型 (总分 0-100)

```
总得分 = 月令分 + min(地支根气分, 40) + min(比劫分 + 偏印分×0.6, 25)
         ± 全局平衡调整
         + 强根加分 (最多+10)
最终 clamp 到 [0, 100]
```

### 4.2 各维度细则

| 维度 | 得分规则 |
|------|---------|
| **月令** | 旺=40, 相=25, 余气=10, 休=0, 囚=0 |
| **地支根气** | 本气=15, 同五行=5, 藏干每个=5；×权重(日支1.2 > 月支1 > 时支0.8 > 年支0.5) → 累加后 cap 40 |
| **天干比劫** | 年/月/时干每有一个比劫 → +10 |
| **天干生扶** | 比劫+10 + 印星+8 |
| **天干克泄耗** | 官杀+8 + 食伤+7 + 财+6 |
| **全局调整** | 克泄耗 > 生扶×2 → -10；生扶 > 克泄耗×2 → +10 |
| **强根加分** | 日支本气根 +5；月令本气且当令 +5 |

### 4.3 等级映射

| 分数 | 等级 | 用神策略 |
|:---:|------|------|
| ≥75 | 极强 | 取克泄耗 |
| ≥60 | 强 | 取克泄耗 |
| ≥48 | 中和偏强 | 取克泄耗 |
| ≥38 | 中和 | 均衡+调候 |
| ≥28 | 中和偏弱 | 取生扶 |
| ≥15 | 弱 | 取生扶 |
| <15 | 极弱 | 取生扶（或从格） |

## 5. 用神忌神

> 实现位置: `annotation/yongShen.ts`

### 5.1 取用规则

| 日主状态 | 用神（喜） | 忌神 | 逻辑 |
|---------|-----------|------|------|
| 身强 | 克泄耗（官杀+食伤+财） | 生扶（印+比劫） | 制强求平衡 |
| 身弱 | 生扶（印+比劫） | 克泄耗（官杀+食伤+财） | 扶弱求平衡 |
| 中和 | 日主 + 印星 | 无明确忌 | 以调候为先 |

### 5.2 调候规则

| 月令 | 调候五行 | 说明 |
|------|:---:|------|
| 巳/午/未 (夏) | 水 | 火炎土燥，喜水润局 |
| 亥/子/丑 (冬) | 火 | 水冷金寒，喜火暖局 |
| 申/酉/戌 (秋) + 日主金 | 火 | 金旺喜火炼 |

**注意**: 调候五行自动追加到用神列表中。

### 5.3 全局偏向调整

- 统计全命局（四柱干支+藏干）各五行出现次数
- 判断全局主导五行和最弱五行
- 用于辅助修正用神忌神选择

## 6. 格局判断

> 实现位置: `annotation/pattern.ts`

### 6.1 判断流程

```
月令藏干是否透出天干?
  ├── 是 → 正格（取第一个透出藏干的十神为格局名）
  └── 否 → 取月令本气对应十神为格局名（隐性格局）

日主极弱?
  ├── 是 → 检查从格（从杀/从财/从儿/从势）
  └── 否 → 检查特殊格局（建禄/阳刃）
```

### 6.2 正格格局高低

| 等级 | 格局类型 |
|:---:|------|
| 上等 | 正官格、正印格、食神格、正财格 |
| 中等 | 偏财格、伤官格、偏官格 |
| 平 | 建禄格（比肩/劫财） |

### 6.3 从格条件

| 从格 | 条件 | 等级 |
|------|------|:---:|
| 从杀格 | 日主极弱 + 官杀≥4个 | 上等 |
| 从财格 | 日主极弱 + 财星≥4个 | 中等 |
| 从儿格 | 日主极弱 + 食伤≥4个 | 中等 |
| 从势格 | 日主极弱 + 非日主五行主导 | 平 |

### 6.4 特殊格局

| 格局 | 条件 | 等级 |
|------|------|:---:|
| 建禄格 | 日干坐禄（甲寅/乙卯/丙巳/丁午/戊巳/己午/庚申/辛酉/壬亥/癸子） | 中等 |
| 阳刃格 | 日干坐帝旺（甲卯/乙寅/丙午/丁巳/戊午/己巳/庚酉/辛申/壬子/癸亥） | 平 |

## 7. 大运流年

> 实现位置: `annotation/luckAnalysis.ts`

### 7.1 大运好坏判断

```
天干五行 = 用神 → +1.5 分
天干五行 = 忌神 → -1 分
地支五行 = 用神 → +1 分
地支五行 = 忌神 → -1 分

总分 ≥ 2 → 佳
总分 ≤ -2 → 不佳
其他 → 平
```

### 7.2 流年太岁分析（6种）

| 关系 | 条件 | 提示 |
|------|------|------|
| **值太岁** | 流年支 = 年支 | 本命年，宜低调行事 |
| **冲太岁** | 流年支 冲 年支 | 变动较大，注意人际关系 |
| **合太岁** | 流年支 合 年支 | 人缘佳，适合合作 |
| **刑太岁** | 流年支 刑 年支 | 防口舌是非和法律问题 |
| **破太岁** | 流年支 破 年支 | 防小人暗中破坏 |
| **害太岁** | 流年支 害 年支 | 防背后中伤和慢性病 |

### 7.3 人生关键节点

| 类型 | 检测方式 |
|------|---------|
| 本命年 | 每12年 |
| 换大运 | 各步大运起始年龄 |
| 冲太岁年 | 年支被冲的各年 |
| 刑太岁年 | 年支被刑的各年 |
| 破太岁年 | 年支被破的各年 |
| 害太岁年 | 年支被害的各年 |

- 按年龄去重，返回前 10 个节点

## 8. 专题批注

> 实现位置: `annotation/specialTopics.ts`

### 8.1 性格

- 日主五行特性（基础）
- 十神主导性格（按数量取前 3 个十神）
- 日支五行特性

### 8.2 事业

- 十神统计判断（官杀≥3 → 体制内；印≥3 → 学术；食伤≥3 → 创意；财≥3 → 经商）
- 特殊组合：官印相生、食伤生财
- 日主五行 → 行业方向

### 8.3 财运

- 正财/偏财数量 → 收入模式
- 食伤生财 → 技术致富
- 比劫多 + 财星 → 破财风险

### 8.4 婚姻

| 性别 | 判断依据 | 
|:---:|------|
| 男 | 财星（正财+偏财）数量，财多→异性缘佳，财无→晚婚 |
| 女 | 官杀（正官+偏官）数量，官杀混杂→选择多，官无→晚婚 |

**配偶宫（日支）分析（6种）**:

| 关系 | 条件 | 含义 |
|------|------|------|
| 逢冲 | 月支或时支 冲 日支 | 聚少离多，需用心经营 |
| 逢合 | 月支或时支 合 日支 | 伴侣人缘佳，需沟通维系 |
| 逢刑 | 月支或时支 刑 日支 | 多摩擦，需互相包容 |
| 被害 | 月支或时支 害 日支 | 易受外界干扰 |
| 被破 | 月支或时支 破 日支 | 感情基础不稳固 |
| 空亡 | 日支落入空亡 | 缘分较淡，宜晚婚 |

### 8.5 健康

- 五行缺失 → 对应器官需保养（木=肝胆, 火=心血管, 土=脾胃, 金=肺, 水=肾）
- 五行过旺（≥5个） → 对应器官易有负担
- 日主偏弱 → 需增强体质

### 8.6 子女

| 性别 | 判断依据 |
|:---:|------|
| 女 | 食伤（食神+伤官）数量 |
| 男 | 官杀（正官+偏官）数量 |

- 时柱为子女宫，附五行分析

## 9. 批注引擎流水线

> 入口: `generateAnnotation(bazi: BaZiResult) → AnnotationResult`

### 9.1 输出数据结构

```
AnnotationResult
├── overview          总览（日主、强弱、格局、用神忌神）
├── strengthAnalysis  日主强弱（分数+原因+各维度分）
├── yongShen          用神忌神（五行列表+理由+调候）
├── wuXingBalance     五行平衡（各五行等级+补益建议）
├── shiShenProfile    十神概况（各十神数量+位置）
├── patternAnalysis   格局分析（类型+成格条件+高低+描述）
├── luckAnalysis
│   ├── daYunList     大运列表（干支+解读+好坏）
│   ├── currentYear   流年分析（解读+关注要点）
│   └── milestones    人生节点（年龄+事件+原因）
├── branchRelations   地支关系（概要+刑冲破害合列表+空亡+合局）
├── specialTopics
│   ├── personality   性格
│   ├── career        事业
│   ├── wealth        财运
│   ├── marriage      婚姻
│   ├── health        健康
│   └── children      子女
└── comprehensiveAdvice  综合建议（最多8条）
```

### 9.2 执行顺序

```
1. analyzeDayMasterStrength    → 日主强弱
2. analyzeYongShen             → 用神忌神（依赖步骤1的等级）
3. analyzePattern              → 格局（依赖步骤1的等级）
4. analyzeWuXingBalance        → 五行平衡
5. analyzeShiShenProfile       → 十神概况
6. analyzeDaYun                → 大运（依赖步骤2的用神忌神）
7. analyzeCurrentYear          → 流年（依赖步骤2的用神忌神）
8. analyzeMilestones           → 人生节点
9. analyzeBranchRelations       → 地支关系
10. analyzeSpecialTopics       → 专题批注（含所有专题子分析）
11. generateOverview           → 总览
12. generateAdvice             → 综合建议（依赖步骤2的用神忌神）
```
