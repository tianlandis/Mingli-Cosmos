§3 防幻觉 L3 护栏体系

L1 — System Prompt 防越权（8 条规则）

规则说明
核心定位解盘者非排盘者，禁止自行推算
工具授权可使用系统工具查询验证
规则0严禁私自排盘 / 取格 / 推八字
规则1数据锁定：回答只能基于命盘数据
规则2禁止绝对化断言词
规则3安全边界：禁止医疗/法律/投资建议
规则4表达风格：平和客观 + 免责声明
规则5话题边界：只回答命理相关
