### **一、月令定盘**

#### **1. 四正月（子、午、卯、酉）**

* **规则**：直接取地支本气的十神定格局。无需考虑透干。
* **示例**：甲日主生于酉月（酉藏辛，正官）→ **正官格**。

#### **2. 杂气月（辰、戌、丑、未、寅、申、巳、亥）**

* **规则**：严格按藏干天数分段判定。必须结合透干规则。

#### **3. 亥月特规**

* **藏干分段**：戊 (2天) → 甲 (7天) → 壬 (21天)。
* **特别备注**：受地球公转“盈缩差”影响，亥月“戊土2天”在现代应用中可能已基本失效。**若出生日落在戊土区间，或用传统规则取格结果与事实严重不符，建议优先考虑取甲木或壬水进行事实验证。**

***

### **二、透干取用（第一优先级规则）**

**核心原则**：**月令藏干“有象先取象”，无论藏干为何，凡透出即取格。比劫也不例外。**

1. **触发条件**：月令所藏的天干之一，出现在年、月、时任一柱的天干上。
2. **取格顺序（优先级）**：
   * **月干 > 年干 > 时干**。
3. **匹配规则**：天干上的字**必须**是该月令的**藏干之一**。否则不算透干。
   * **示例**：丑月（藏癸、辛、己）。年干庚金，非丑月藏干，**不能**取为正官格。
4. **比劫透干处理**：
   * **若月干透出**比肩或劫财，**直接取格**。此时不用辰戌丑未的杂气特殊规则。
     * **示例**：甲日主，寅月，月干为甲（比肩）。取为**比肩格**（此时已触发禄格成立条件，见下文）。

***

### **三、不透则用“月令分金”（第二优先级规则）**

**触发条件**：月令所有藏干均未透出天干。

这是一个精密的三步算法：

#### **第一步：定尺与输入**

1. **定尺**：调用对应月支的**分金刻度尺**（藏干天数表）。
2. **计算起运天数**：以节气为界，计算出身日到最近节气的天数。
   * **大运顺排（阳男阴女）**：**顺数**到下一个节气。
   * **大运逆排（阴男阳女）**：**逆数**到上一个节气。
   * **换算规则**：3天 = 1岁起运；1天 = 4个月；1时辰 ≈ 10天。
3. **方向判定**：
   * **阳男阴女（大运顺排）**：分金**逆数**（逆着刻度尺的方向数）。
   * **阴男阳女（大运逆排）**：分金**顺数**（顺着刻度尺的方向数）。

#### **第二步：定位与取格**

1. **定位**：
   * **顺数**：直接用**起运天数**，从刻度尺开头往后数。
   * **逆数**：用 **30 - 起运天数**，从刻度尺末尾往前数。
2. **取格**：根据最终指向的数值，确定对应哪个藏干，以此与日干十神关系**定格局**。

#### **第三步：特殊处理（比劫不成格 & 禄刃破例）**

**1. 规则：比劫不成格**

* **适用场景**：定位结果落在比肩或劫财上。
* **处理原则**：此比劫**不能**直接取为格局。需**按分金方向（即行运方向）**，再**进一位或退一位**，取相邻的财、官、印、食等十神为格。

**2. 例外：禄格 & 羊刃格（优先级高于比劫不成格的通用规则）**

* **前提**：分金结果落在**比肩**或**劫财**上。
* **破例条件**：
  * **禄格**：当前月支为**四长生月（寅、申、巳、亥）**，且日干在此月**得禄**。
    * **示例**：甲日主，寅月第20天以后，不透干，分金逆算后定位在**甲**（比肩）。寅为甲之临官（禄），故破例立为**禄格**。
  * **羊刃格**：当前月支为**四正位（子、午、卯、酉）**，且日干在此月为**羊刃**。
    * **示例**：庚日主，酉月第30天，不透干，分金定位在**辛**（劫财）。酉为庚之帝旺（羊刃），故破例立为**羊刃格**。
* **生效路径**：若分金结果、月份和日干同时满足上述条件，**直接破例成立禄格或羊刃格**，不再执行“比劫不成格”的进退位规则。

### **四、大运与流年基本法则**

1. **大运**：**重地支，轻天干**。主要看大运地支与格局、日干的**十二长生关系、生克制化关系**。
2. **流年**：**重天干，轻地支**。流年天干主要看其与格局、大运的**合化关系**（如合去、合坏、合动）以及是否构成“**伤官见官**”等关键组合。流年地支主要论与局中地支的**刑、冲、合、害**及神煞关系。
3. **吉凶判断标准**：**先看克，后看生**。优先判断大运、流年对格局是**维护**（生扶用神）还是**破坏**（克制用神/引动忌神）。
   * **吉格**（财官印食等）：喜生扶，忌破坏。
   * **凶格**（杀伤枭刃等）：喜制化，忌激化。

### **五、吉凶格分类与基本喜忌**

| **格局类型**              | **吉凶属性**    | **基本喜忌**                                                     |
| --------------------- | ----------- | ------------------------------------------------------------ |
| **正官、正印、偏印、食神、正财、偏财** | **吉格（喜生扶）** | 喜大运、流年生扶该十神，最忌被克制、混杂、合坏。                                     |
| **七杀、伤官、羊刃**          | **凶格（喜制化）** | **七杀/伤官**：喜见印（化）或食神/正印（制）。&#xA; **羊刃**：喜见七杀（驾），或食伤（泄），或印（化）。 |

### **六、终极决策树**

```
    是否四正月（子午卯酉）？
        |
        ├── 是 → 直接取本气十神定格局。
        |
        └── 否 → 检查月令藏干是否透出天干？
                |
                ├── 是 → 取透干者（月干 > 年干 > 时干）。至此结束。
                |
                └── 否 → 启动“月令分金”算法：
                        |
                        ├── 第一步：根据性别/年干/节气计算“起运天数”。
                        ├── 第二步：调用“藏干天数刻度尺”进行顺数/逆数定位。
                        └── 第三步：检查定位结果。
                                |
                                ├── 结果比劫？ → 检查是否为“禄格”或“羊刃格”破例条件？
                                |       |
                                |       ├── 是 → 立“禄格”或“羊刃格”。
                                |       └── 否 → 按分金方向，进/退一位，取相邻非比劫十神为格。
                                |
                                └── 结果非比劫 → 直接取该藏干十神定格局。
```
