# 数字命理推演引擎 · Mingli-Cosmos

> **正宗子平法命理排盘引擎** — 传统规则为骨，现代 AI 为翼

基于 **React 19 + TypeScript + Vite + Tailwind CSS 4 + Hono** 构建的全栈新中式八字命理 Web 应用。核心引擎以 **Python MCP（tengods）为算法权威参考**，严格遵循传统子平法理论；后端配备 **AI 流式对话、管理后台、知识字典引擎**，已部署至 Linux VPS 生产环境。

## 🎯 为什么这是"真传"

市面上大多数八字产品用简表查月、固定藏干、笼统格局，**这里不是**：

| 特性 | 本项目 | 市面常见 |
|------|:---:|:---:|
| **节气换月** | ✅ 精确到节气交节时刻 | ❌ 固定日期换月 |
| **地支藏干** | ✅ 完整本气/中气/余气（Python MCP 对齐） | ❌ 仅取本气或简化 |
| **月令分金** | ✅ 藏干天数刻度尺精确到天（动态高精度校准） | ❌ 不支持 |
| **格局判断** | ✅ 四正/透干/分金三路径（多维联合防偏校验） | ❌ 简单正格枚举 |
| **起运天数** | ✅ 分金依赖精确计算 | ❌ 粗略估算 |
| **十神推导** | ✅ 以日主为中心完整推导 | ❌ 部分缺失 |
| **大运排法** | ✅ 阴阳顺逆 + 节气换运 | ❌ 简化排法 |
| **神   煞** | ✅ 8 种经典神煞 + MBTI 人格映射 | ❌ 缺失或不准确 |
| **AI 解读** | ✅ Qwen3.5-122B-A10B + SSE 流式对话 | ❌ 无 AI 或简单模板 |
| **管理后台** | ✅ OpenClaw 风格驾驶舱 + DB 热配置 | ❌ 无 |

> **核心理念**：计算准确 + 规则透明 + 文化尊重。每条批注可解释、可溯源、可验证。

### 🔬 算法权威基准

引擎不依赖"黑盒"逻辑，所有推演均基于 Python MCP (tengods_mcp) 定义的严苛标准对齐：

- **格局取用**：四正取格 + 透干优先级多维校验，有效规避比劫干扰
- **分金迭代**：内嵌标准分金算法，支持正反向精确迭代
- **神煞映射**：基于标准命理规则 + MBTI 现代人格映射双重验证

详见：[`docs/arch/ALGORITHM-AUTHORITY.md`](./docs/arch/ALGORITHM-AUTHORITY.md)
